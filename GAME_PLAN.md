# План разработки 3D Шутера

## Обзор проекта

**Цель**: Создать браузерный 3D шутер от первого лица (FPS)

**Технология**: Three.js + Next.js (браузерная игра без Electron)

---

## ✅ Этап 1: Базовая структура (ВЫПОЛНЕНО)

### Реализовано:
- [x] Three.js сцена с освещением и тенями
- [x] Камера от первого лица
- [x] Управление мышью (Pointer Lock API)
- [x] WASD движение + прыжок (Space)
- [x] Гравитация и коллизии
- [x] Уровень с полом, стенами и укрытиями
- [x] Оружие и стрельба (рейкастинг)
- [x] Враги с простым ИИ (преследование)
- [x] HUD (здоровье, патроны, прицел, счёт)
- [x] Экран Game Over

---

## Управление

| Клавиша | Действие |
|---------|----------|
| WASD | Движение |
| Мышь | Обзор |
| ЛКМ | Стрельба |
| Space | Прыжок |
| R | Перезарядка |
